## üì¶ Modules

### 1. [Decorators](./decorators/)

Master Python decorators for adding functionality to functions and classes.

**Topics covered:**
- Basic function decorators
- Decorators with arguments
- Memoization and caching
- Timing decorators
- Class decorators (Singleton pattern)
- Input validation decorators

### 2. [Generators](./generators/)

Learn memory-efficient data processing with generators and iterators.

**Topics covered:**
- Basic generator functions
- Generator expressions
- Infinite generators
- Generator pipelines
- Custom iterator classes
- `yield from` delegation
- Large file processing

### 3. [Context Managers](./context_managers/)

Understand resource management and the `with` statement.

**Topics covered:**
- Basic context manager protocol
- Timer context managers
- `@contextmanager` decorator
- Exception handling
- Transaction patterns
- Reentrant locks
- ExitStack for dynamic management

### 4. [Lambda Functions](./lambda_functions/)

Learn Python's concise anonymous functions.

**Topics covered:**
- Basic lambda syntax
- Lambda with map(), filter(), sorted()
- Custom sorting with lambdas
- Dictionary operations
- Lambda vs regular functions

### 5. [List Comprehensions](./list_comprehensions/)

Master Python's elegant way to create lists.

**Topics covered:**
- Basic list comprehension syntax
- Filtering with conditions
- If-else expressions
- Nested comprehensions
- String processing

### 6. [String Formatting](./string_formatting/)

Format strings like a pro with f-strings and more.

**Topics covered:**
- F-strings (formatted string literals)
- Number formatting
- Text alignment and padding
- The format() method
- Practical formatting patterns

## üöÄ Getting Started

### Prerequisites

- Python 3.7 or higher
- Basic understanding of Python syntax
- Familiarity with object-oriented programming

### Installation

Clone the repository:

```bash
git clone https://github.com/GIL794/Simply-Python.git
cd Simply-Python
```

No additional dependencies are required - all examples use Python's standard library.

## üíª Usage

Each module can be run independently:

```bash
# Run decorators examples
python decorators/decorators.py

# Run generators examples
python generators/generators.py

# Run context managers examples
python context_managers/context_managers.py

# Run lambda functions examples
python lambda_functions/lambda_functions.py

# Run list comprehensions examples
python list_comprehensions/list_comprehensions.py

# Run string formatting examples
python string_formatting/string_formatting.py
```

For learning, we recommend:

1. Read the module's README first
2. Run the demonstration script
3. Study the source code and comments
4. Experiment by modifying the examples
